Yebba Yennebiebe

An VR videogame made with Unity

If the video doesn't work, click HERE

About the videogame

Yebba Yennebiebe is a VR videogame made with Unity. You are a wizard who has to face hordes of monsters and you will have to use your hands and cast spells to eliminate them. There is the option to teleport and there is a table with the different spells. In each hand you can have a different spell, there are three types: Fire, Ice (slows down the monster) and Electric (paralyses the monster).

About the creation of the videogame

This project was done together with three other classmates. This was the first project we did in VR, and it was a great experience. Although it is a bit more complicated to test everything you implement and you lose some time between putting on the glasses, testing it, etc., it was a really fun project and I think it turned out to be a great game.

Third Party Libraries

This videogame was made using only Unity.
