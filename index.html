About me

My name is Lucas Pérez and I'm a game developer from Barcelona. I really like to develop video games and keep improving my C++ skills. I'm very interested in the game engines, how they work internally, and also the rendering and shaders part. My main goal is to be able to work in the videogame industry in one of these fields.

At this moment I'm studying a bachelor degree in Video Game Design and Development in CITM university. We mainly learn how to program with C++ but we also learn proper techniques about game design and art.

In this portfolio you can see all the projects I've done and you can also see my github.
